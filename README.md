# UNDERPLOT
Code to create the UNDERPLOT dataset (Bruelheide et al. 2026, Scientific Data)
Step1 : Taxonomic harmonization
In total 15 different sources were taxonomically harmonized (see comments at the beginning of the file).
These sources are available online (see URLs, DOIs and references in the code) or are available at request.
Names were matched against the sPlot 4.1 backbone (available at request), which is based on WFO (world flora online).
As a result, taxon names were harmonized.

Step 2: Deriving Belowground Persistence Type (BPT) and Clonal Spread Index (CSI). 
In addition, clonal growth organ (CGO) names were harmonized.
  
Step 3: The different databases are joined, also aggregating multiple trait values per taxon to a single value

Step 4: The joined database is combined with TRY and inconsistencies were removed. 
Names of growth forms were harmonized. In addition, information on single species were added (all summarized in the
Supplementary Information S1 in the Scientific Data publication).

The final database is available at the iDiv data portal under
https://doi.org/10.25829/idiv.3610-56acy9
